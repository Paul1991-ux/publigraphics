# publigraphics 0.1.0

* Initial release.
* BibTeX/RIS/CSV parsing with standardised schema.
* Automatic classification into 12 canonical production types.
* Thematic word clouds (TF-IDF + LDA topic modelling).
* Co-authorship network visualisation.
* Geographic mapping of seminar interventions.
* Gantt charts for funded research projects.
* AI-generated narrative summaries via Anthropic Claude API.
* Premium HTML and PDF notebook generation.
* Bilingual interface (French / English).
* Demo data: Esther Duflo (MIT, Nobel Prize 2019).
